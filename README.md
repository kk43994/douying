# 抖影 Douying

面向短视频增长的本地工具：抖音单视频/账号分析 + 视频口播文案提取 + 脚本生成。

## 功能概览
- 单视频：解析链接元数据、提取视频口播文案（无字幕时点「提取视频文案」）、基于口播与数据做深度拆解（只输出中文、Markdown 排版）
- 账号：账号基础信息分析 + 账号/视频工具箱（前端占位，可导出/复制报告、Top10 样本展示）
- 生成：脚本生成、模板、人物设定与产品库
- 设置：分析模型配置、文案提取 API 配置、语音识别配置、流式输出开关

## 快速开始（开发模式）
1) 安装依赖：`npm install`
2) 启动：`npm run dev`
3) 打开：`http://127.0.0.1:3000/`

## 一键启动/关闭（Windows）
- 启动：双击 `启动平台.bat`
- 关闭：双击 `关闭平台.bat`
- 重启：双击 `重启平台.bat`
- 常见问题：见 `使用说明.md:1`

## 配置说明
- 分析模型配置：在「设置」里配置（默认按云雾 OpenAI 兼容接口）
  - 也可用环境变量（可选）：复制 `.env.example` 为 `.env.local` 后填写
- 语音识别配置（用于视频口播文案提取）：默认豆包语音（也保留阿里百炼 Paraformer 作为备选，不默认启用）
- 文案提取 API 配置：独立于“分析”配置，当前作为“从链接直取口播文案”的备用方案（可能需要联网插件）

## 打包发布（发给别人）
双击 `打包发布.bat`，会在 `release/` 下生成 zip 压缩包。

## 安全提示
- 不要把 `.env.local` 提交到 GitHub（本项目已在 `.gitignore` 忽略）。
